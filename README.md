# بوت رفع الملفات إلى Top4top

هذا البوت يتيح لك رفع الصور والبرامج والملفات مباشرة إلى موقع top4top.io عبر تلجرام.

## المميزات:
- دعم رفع الصور والفيديوهات والملفات (Documents).
- الحصول على روابط التحميل مباشرة في تلجرام.
- التحكم الكامل عبر واجهة التلجرام.

## كيفية التشغيل:
1. قم بالحصول على توكن من [BotFather](https://t.me/botfather).
2. افتح ملف `.env` وقم بوضع التوكن الخاص بك مكان `YOUR_BOT_TOKEN_HERE`.
3. قم بتثبيت المكتبات المطلوبة (إذا لم تكن مثبتة):
   ```bash
   npm install
   ```
4. ابدأ تشغيل البوت:
   ```bash
   npm start
   ```

## التشغيل على Vercel (Webhooks):
إذا كنت تريد تشغيل البوت 24/7 دون الحاجة لجهازك الشخصي، يمكنك رفعه على Vercel:

1. قم برفع المجلد إلى مستودع (Repository) على GitHub.
2. اربط المستودع بـ Vercel.
3. أضف متغير البيئة `TELEGRAM_BOT_TOKEN` في إعدادات Vercel.
4. بعد النشر، ستحصل على رابط (مثلاً: `https://my-bot.vercel.app`).
5. قم بتفعيل الـ Webhook بتشغيل هذا الأمر في جهازك (مرة واحدة فقط):
   ```bash
   node set-webhook.js https://your-app.vercel.app
   ```
   *(استبدل الرابط برابط التطبيق الخاص بك على Vercel)*.

## ملاحظة هامة لـ Vercel:
- Vercel لديه وقت محدد لتنفيذ الدوال (حوالي 10-30 ثانية). إذا كان الملف كبيراً جداً، قد يفشل الرفع.
- يفضل استخدام Vercel للصور والملفات الصغيرة والمتوسطة.
